# Multi-Area App
A comprehensive multi-feature application implementing a scalable structure.

## Features
- Home, AI Assistant, Search screens
- Gaming Area with Play Games, Live Gaming, and Leaderboard
- Shopping Area with Products, Cart, and Orders
- Music Area with Songs and Live Music
- Upload, Camera, Channels, Live Stream
- Profile and Settings

## Tech Stack
- Flutter
- Material 3
- Responsive Drawer Navigation

## Setup
Run the app using standard flutter commands:
`flutter run`
